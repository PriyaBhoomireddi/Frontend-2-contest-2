パーツに対して使用することが予想される最小厚さを指定します。 機能要件を満たすために、壁は可能な限り薄くします。 標準的な壁の厚さは、パーツのサイズ、パーツの適用方法、パーツの設定、および使用する合金によって異なります。

金属の流れを良くして鋳物を完全に充填するために、鋳物の周囲の壁の厚さの変位を最小限に抑えます。 壁の厚さが異なると、鋳物の各壁に沿った冷却速度が異なってしまい、パーツの寸法にずれが生じる可能性があります。

ソルバは、最小厚さをその他の拘束や目標(勾配角度、取り出し方向、剛性、安全率、ターゲット質量など)と組み合わせて使用します。 場合によっては、他の拘束を満たすことができるように、最終的な厚さが設定されている最小厚さの値を下回る可能性があります。
