Gibt einen Durchmesser des kleinsten Schnittwerkzeugs an, das für eine Fräsoperation berücksichtigt wird.
Werkzeugdurchmesser
Schneidenlänge

Die Größen von Ergebniselementen wie z. B. Taschen, Nuten und Abrundungen hängen vom Mindestwerkzeugdurchmesser ab. Auch wenn Sie mehrere Werkzeuge im Fertigungsprozess verwenden, geben Sie den Durchmesser des kleinsten Werkzeugs an, dessen Einsatz Sie planen.
