Orienta lo strumento nella direzione parallela all'asse Z della triade e rivolta verso l'estremità negativa dell'asse Z.

I pulsanti di direzione strumento permettono di impostare gli orientamenti utensile per un'operazione di fresatura. La direzione dello strumento è orientata con riferimento al sistema di coordinate globali.

È possibile selezionare diverse direzioni strumento per una configurazione di fresatura per permettere la lavorazione a macchina a più lati.
