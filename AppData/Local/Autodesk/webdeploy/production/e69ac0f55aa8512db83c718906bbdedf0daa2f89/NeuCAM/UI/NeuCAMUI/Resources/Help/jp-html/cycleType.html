ドリル サイクルのタイプを選択すると、ドリル操作に使用できるパラメータに変更します。 業界共通の G コードで表示されます。

  • ドリル: 深さに送り、早送り退出 - 工具直径の 3 倍未満の深さの穴の場合。(G81)
  • ざぐり: 深さに送り、停留して、早送り退出 - 停留を使用して穴のボトム仕上げを改善します。(G82)
  • チップ切断: 複数のペックを使用して、チップを切断したり穴にクーラントを注入させながら工具を定期的に退避させます。工具直径の 3 倍を超える深さの穴の場合。(G73)
  • 深穴ドリル: チップ切断に似ていますが、穴の外に退避してチップを除去し、穴をクーラントであふれさせます。ペック ドリルとも言われます。(G83)
  • 貫通: 穴を貫通するまでの送りを減少したり減速することができます。
  • ガイド深穴ドリル: ガン ドリルには単一の切削エッジがあります。ガイド パッドによって穴が磨かれ、正確な直径のまっすぐで深い穴を作成します。
  • タップ: 選択されている工具に応じて、右ねじまたは左ねじをタップします。
  • 左タップ: 深さに送り、主軸を反転させ、送り退出 - ねじ山をカットするために穴に入るときに反時計回りに回転します。(G74)
  • 右タップ: 深さに送り、主軸を反転させ、送り退出 - ねじ山をカットするために穴に入るときに計回りに回転します。(G84)
  • チップ切断でタッピング: 最終深さに達するまで徐々に深くなるように複数回出し入れを繰り返します。
  • リーマ: 深さに送り、最終深さに達した後すぐに送り退出します。(G85)
  • ボア: 深さに送り、停留し、送り退出します。(G86)
  • ボア停止: 深さに送り、主軸を停止し、早送り退出します。(G87)
  • ファイン ボア: 深さに送り、主軸を停止し、壁から移動して、早送り退出します。(G76)
  • バック ボア: 既存の穴のバックからボア加工します。(G77)
  • 円形ポケット ミル: 円形ポケットのカスタム サイクルを呼び出すのに使用することができます。
  • ボア ミル: らせんボア ミルのカスタム サイクルを呼び出すのに使用することができます。
  • ねじ切りミル: らせんねじ切りミルのカスタム サイクルを呼び出すのに使用することができます。
  • プローブ: マシンからプローブ工具のマクロを呼び出すのに使用されます。カスタム ポストプロセッサが必要です。[セットアップ]プルダウン メニューでより高度なプローブの機能を参照してください。
