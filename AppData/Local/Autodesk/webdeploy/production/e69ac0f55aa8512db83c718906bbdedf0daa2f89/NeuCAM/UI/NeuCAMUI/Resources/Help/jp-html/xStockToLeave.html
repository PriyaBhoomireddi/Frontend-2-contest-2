径方向に残す材料の量をコントロールします。
パーツの内径または外径に残すストックのために使用します。

X 軸方向のストック 1 mm

X 軸方向のストック 1 mm

X 軸方向のストック 0.0

X 軸および Z 軸方向のストック -1 mm

正確な水平ではないサーフェスでは、プログラムは軸方向の仕上げ代の値(壁)と径方向の仕上げ代の値の間を補間します。これらのサーフェス上の径方向に残されるストックは、サーフェスの勾配や軸方向仕上げ代の値によっては、指定した値と異なる場合があります。

X 軸仕上げ代 の値を変更すると、Z 軸仕上げ代 が同じ値に自動的に設定されます。別の軸方向の仕上げ代の値を手動で入力することができます。

別の工具で粗取りや仕上げを行う場合は、次の仕上げ操作のために仕上げ代の量に粗取りの操作を設定します。

負の仕上げ代 負の値を使用することができます。 これにより、工具はモデル形状のエッジを越えて材料を機械加工します。

注: 負の仕上げ代の最大値は工具先端の半径より小さくする必要があります。
インサートの先端の半径が 0.032 の場合、負の仕上げ代の最大値は -0.032 となります。
ツールの理論的な先端を越えて補正することはできません。
