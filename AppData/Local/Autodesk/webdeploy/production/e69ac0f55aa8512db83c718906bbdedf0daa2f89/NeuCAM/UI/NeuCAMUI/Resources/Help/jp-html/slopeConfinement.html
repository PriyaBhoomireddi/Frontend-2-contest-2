傾斜 角度での制限は、形状 タブにある 傾斜開始角度 と 傾斜終了角度 のパラメータで指定します。角度は 0 度(水平)から 90 度(垂直)の間で定義します。

傾斜開始角度 と 傾斜終了角度 に入力されている値以上の領域のみが加工されます。

ほとんどの 3D 仕上げ加工方法が傾斜角度での制限をサポートしています。傾斜角度での制限の用法の 1 つとして、選択されているツールパス加工方法が最も適している角度に制限するというものがあります。たとえば、走査線仕上げ は緩やかな領域の方が適しており、一方で 等高線仕上げ は急な領域の方が適しています。
