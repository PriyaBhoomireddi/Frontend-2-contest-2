Se selezionata, la selezione di una faccia individuerà tutte le facce sullo stesso piano, da tutte le parti all'interno della boundary del grezzo.

Selezionare una faccia qualsiasi su una parte qualsiasi per una corrispondenza.

NOTA: utilizzata per filtrare le boundary in base alla profondità della faccia. Utilizzare il filtro Linee chiuse per individuare tutte le boundary interne o esterne.
