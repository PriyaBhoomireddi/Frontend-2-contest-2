Il tipo consente di classificare i cluster in base all'angolo tra i triangoli perpendicolari lungo i loro bordi. Se più del 75% della lunghezza del bordo di un cluster è formata da triangoli confinanti con perpendicolari convergenti, un cluster è considerato concavo. Se più del 75% dei triangoli confinanti hanno perpendicolari divergenti, il cluster è considerato convesso. I casi intermedi non sono considerati.
