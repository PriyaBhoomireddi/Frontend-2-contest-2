Grazie a questa opzione è possibile controllare se i triangoli di un cluster devono essere supportati con la rispettiva azione anche quando un'azione precedente nello script ha già applicato il supporto agli stessi triangoli. Sovrapponendo le aree supportate in questo modo, è possibile creare una densità di supporto a gradini che è più intensa dove l'angolo di downskin è, ad esempio, particolarmente superficiale.
