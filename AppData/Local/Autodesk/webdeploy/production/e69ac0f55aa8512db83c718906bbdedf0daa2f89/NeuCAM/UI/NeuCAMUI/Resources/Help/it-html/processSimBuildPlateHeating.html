Specifica se la piastra di costruzione è riscaldata o meno.

  • Nessuno: la piastra di costruzione non è riscaldata o la temperatura non è nota.
  • Iniziale: la piastra di costruzione è pre-riscaldata ad una temperatura specifica.
  • Controllato: la base della piastra di costruzione mantiene una temperatura costante durante l'intero processo di produzione.
