Le menu déroulant Origine propose les options suivantes pour définir l'origine du trièdre:

Régler origine SCG: utilise l'origine du système de coordonnées de la pièce de travail de la configuration actuelle pour définir l'origine de l'outil.

'''Point de surface (extension d'usinage)''': utilise le point de surface sélectionné pour l'origine du trièdre.

Origine du modèle: utilise l'origine du système de coordonnées de la pièce actuelle pour définir l'origine de l'outil.

Point sélectionné: permet de sélectionner un sommet ou une arête pour définir l'origine du trièdre.

Point sur le brut: permet de sélectionner un point sur la zone de délimitation du brut pour définir l'origine du trièdre.

Point sur le modèle: permet de sélectionner un point sur la zone de délimitation du modèle pour définir l'origine du trièdre.
