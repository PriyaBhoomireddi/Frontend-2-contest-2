Specifica l'orientamento dell'asse rotativo.

Il menu a discesa Orientamento fornisce le seguenti opzioni per impostare l'orientamento dell'asse rotativo:

Setup asse X: utilizza l'asse X del sistema di coordinate globali (WCS) del pezzo di lavoro del setup corrente come asse rotativo. Di conseguenza, la direzione utensile è sempre perpendicolare all'asse X.

Setup asse Y: utilizza l'asse Y del sistema di coordinate globali (WCS) del pezzo di lavoro del Setup corrente come asse rotativo. Di conseguenza, la direzione dell'utensile è perpendicolare all'asse Y.

Setup asse Z: utilizza l'asse Z del sistema di coordinate globali (WCS) del pezzo di lavoro del setup corrente come asse rotativo. Di conseguenza, la direzione dell'utensile è perpendicolare all'asse Z.

Orientamento modello - Utilizza il sistema di coordinate (WCS) della parte corrente come l'asse rotativo.

Seleziona asse rotativo - Selezionare un piano rispetto a cui impostare l'asse rotativo perpendicolare. È inoltre possibile selezionare uno spigolo rispetto a cui impostare l'asse rotativo parallelo.

Seleziona sistema di coordinate: imposta un orientamento specifico dell'asse rotativo per questa operazione da un sistema di coordinate definito dall'utente (UCS) nel modello. Utilizza l'origine e l'orientamento del sistema di coordinate esistente. Utilizzare questa opzione se il modello non contiene un punto e un piano adatti per l'operazione.
