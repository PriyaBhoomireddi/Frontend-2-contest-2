ダウンカット対アップカット

ミーリング工具は刃先が最大限の厚みで材料にかみ合い、その後はかみ合いを減らしながら 0 へと戻るような形で材料を貫いて進んでいきます。これをダウンカットといいます。

逆方向で切削を行うと、工具は厚み 0 から始めて最大値まで増加しながら材料をすくい上げます。これをアップカットといいます。

汎用機ではアップカットがよく使われています。汎用機でダウンカットを使うと、リードねじのバックラッシュによって工具がふらつくためです。CNC工作機にはボールねじが使われているので、このような問題は起こりません。

アップカットを使用すると工具と切削表面に摩擦が生じ、材料を硬化させ、発熱し、工具の磨耗が進みます。また、切り粉が仕上がり面をひっかく事により、面の仕上がりも損なわれます。

工具メーカーが特に推奨する材料がある場合を除き、CNC では常にダウンカットを使ってください。ダウンカットを使うと、切削圧と発熱をはるかに低く抑えることができ、表面の仕上がりはよくなり、工具の寿命も長くなります。

注意: アクリルのように破損しやすい材料を使うとき、材料のエッジが破損しないようにダウンカットを使うとよいでしょう。
