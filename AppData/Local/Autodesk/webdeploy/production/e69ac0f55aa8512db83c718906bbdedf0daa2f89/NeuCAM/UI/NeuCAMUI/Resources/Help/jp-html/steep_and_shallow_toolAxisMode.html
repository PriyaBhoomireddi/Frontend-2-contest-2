複合軸加工の操作工具軸を指定します。

垂直 - 工具を指定された工具方向に位置合わせします。

進入と傾斜 - 進入/傾斜角度を指定することにより、工具軸を傾けます。0の進入/傾斜角度は、工具軸が接触する表面に対して常に垂直になるように位置合わせします。進入角度は切削方向に工具を傾けます。傾斜角度は切削方向から工具を横に傾けます。進入/傾斜角度を使用する必要がある場合の例は、工具軸を傾けて、ボール状の工具の中心で切断しないようにするときです。

カーブから - 工具の先端はスケッチされた曲線の反対方向を向きます。

カーブへ - 工具の先端はスケッチされた曲線の方を向きます。

開始点 - 工具の先端はスケッチされた点の反対方向を向きます。

終了点 - 工具の先端はスケッチされた点の方を向きます。

垂直

進入と傾斜 = 0
1 - 移動の方向
2 - 進入角度傾斜方向
3 - 傾斜角度傾斜方向
