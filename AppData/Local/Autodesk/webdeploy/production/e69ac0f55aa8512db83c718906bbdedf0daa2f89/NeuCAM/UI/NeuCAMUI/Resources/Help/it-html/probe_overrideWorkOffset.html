Controlla quale sistema di coordinate determina l'operazione di indagine.

Quando questa opzione è selezionata, l'ispezione è guidata da un sistema di coordinate diverso (sostituzione WCS) rispetto a quello definito nel setup (offset WCS). La sostituzione WCS per questa operazione di ispezione non viene aggiornata, mentre invece viene aggiornato l'offset WCS per le operazioni di lavorazione successive. Questa operazione blocca il sistema di coordinate per l'ispezione.

Esempio di offset WCS (1=G54) aggiornato, ma nessuna modifica per la sostituzione WCS (2=G55) per più parti:

Sostituisci WCS di azionamento selezionato

Quando questa opzione è deselezionata, l'ispezione è guidata dallo stesso sistema di coordinate definito nel setup (offset WCS). L'offset WCS per questa operazione di ispezione e le successive operazioni di lavorazione viene aggiornato.

Esempio di offset WCS (1=G54) aggiornato per più parti:

Sostituisci WCS di azionamento deselezionato
