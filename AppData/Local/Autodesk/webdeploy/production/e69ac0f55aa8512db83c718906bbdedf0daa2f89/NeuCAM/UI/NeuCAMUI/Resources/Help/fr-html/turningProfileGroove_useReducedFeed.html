Activez cette option pour réduire la vitesse d'avance lors du rainurage le long de l'axe X. Les passes de finition sont malgré tout réalisées à la vitesse d'avance d'origine spécifiée dans l'onglet Outils.

Les éléments mis en surbrillance en bleu dans la trajectoire d'outil utilisent la
vitesse réduite spécifiée dans le champ Avance réduite.
