物理パーツ上で実行する検査のタイプ。

  • 距離: 穴の直径やボスの直径、または 2 つの平行なサーフェス間の距離を計測する場合に選択します。距離が選択されている場合、キャンバス内でフィーチャをクリックする必要があります。
  • 合格/不合格: 定義された基準を確認する場合に選択します。
  • テキスト: カスタム テストを確認する場合に選択します。

手動検査の記録時に、手動検査を記録ダイアログで使用できるオプションは、選択した検査のタイプによって異なります。次のようになります:

  • 距離: 計測値を入力します。
  • 合格/不合格: 合格または不合格オプションを選択します。
  • テキスト: 結果ボックスにカスタム テストの結果を入力します。
