退避高さの位置は、移動高さとボトム高さの間での中間高さを設定します。

参照退避高さをプローブ
  • 移動高さ: 移動高さ からの相対オフセット値です
  • ボトム高さ: ボトム高さ からの相対オフセット値です。
  • モデル トップ: モデル トップ からの相対オフセット値です。
  • モデル ボトム: モデル ボトム からの相対オフセット値です。
  • ストック トップ: ストック トップ からの相対オフセット値です。
  • ストック ボトム: ストック ボトム からの相対オフセット値です。
  • 選択: モデル上で選択された 点(頂点)、エッジ、または 面 からの相対オフセット値です。
  • プローブ サーフェス トップ: 選択されたサーフェス最下部のトップからの相対オフセットです。
  • 原点(絶対): セットアップ または指定した操作の 工具方向 で定義された 原点 からの絶対オフセット値です。
