Valeur maximale d'engagement que la trajectoire d'outil adaptative doit conserver.

Elle peut être considérée comme le nombre de recouvrements. Toutefois, l'usinage adaptatif à haute vitesse modifiera le recouvrement afin de réduire la surcharge de l'outil.

Les trajectoires d'outil traditionnelles peuvent surcharger l'outil dans les coins. L'évidement adaptatif accélère la suppression de matière et élimine les pointes d'insertion d'outil qui peuvent casser les outils de découpe.

Charge optimale de 3.0mm

REMARQUE: cette valeur doit être inférieure au rayon de l'outil.
