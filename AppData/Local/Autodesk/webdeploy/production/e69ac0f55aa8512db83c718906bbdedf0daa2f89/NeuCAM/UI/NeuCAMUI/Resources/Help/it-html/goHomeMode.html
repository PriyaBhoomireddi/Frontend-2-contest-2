La posizione Inizio è un valore Z noto relativo al WCS ed è definita nella sezione Sistema di coordinate globali (WCS) della scheda Setup nella finestra di dialogo della strategia.

È possibile forzare l'utensile a spostarsi nella posizione Inizio prima di iniziare l'operazione o al termine della stessa. L'utensile verrà sempre estratto dal materiale grezzo nell'asse X finché non raggiunge l'altezza sicura, quindi si sposterà nella posizione Inizio in Z.

Nota: la posizione Inizio a cui si fa riferimento qui è diversa dalla posizione Inizio G28 o G53 definita nella finestra di dialogo Post-processor con alcuni post. L'impostazione simultanea di entrambi i valori potrebbe creare risultati imprevisti se è la prima volta che si utilizzano il post-processor e la macchina.
