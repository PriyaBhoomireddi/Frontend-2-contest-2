ボトム高さでは、最終加工高さ/深さと工具がストックに降下するときの最も低い深さを決定します。ボトム高さは、トップ より低くなるように設定する必要があります。ボトム高さとそれ以降のオフセット量を組み合わせて使用することで、高さが決定します。

ボトム高さ
  • 移動高さ: 移動高さ からの相対オフセット値です
  • 退避高さ: 退避高さ からの相対オフセット値です。
  • 送り高さ: 送り高さ からの相対オフセット値です。
  • トップ高さ: トップ高さ からの相対オフセット値です。
  • モデル トップ: モデル トップ からの相対オフセット値です。
  • モデル ボトム: モデル ボトム からの相対オフセット値です。
  • ストック トップ: ストック トップ からの相対オフセット値です。
  • ストック ボトム: ストック ボトム からの相対オフセット値です。
  • 選択された輪郭: モデル上で選択された 輪郭 からの相対オフセット値です。
  • 選択: モデル上で選択された 点(頂点)、エッジ、または 面 からの相対オフセット値です。
  • 原点(絶対): セットアップ または指定した操作の 工具方向 で定義された 原点 からの絶対オフセット値です。
  • 面取り幅: パラメータに入力された面取り幅を作るのに十分な深さになるよう工具で穴あけします。入力するパラメータは、工具の面取りの幅を超えてはいけません。算出される高さ方向のオフセット量は、工具のパラメータ(直径、先端径、刃先角度)と穴の径によって変わります。円柱形の面、円、円弧を選択することができます。
  • 面取り直径: 新しい穴の直径として入力したパラメータと等しい直径です。 したがって、入力するパラメータは、工具の直径を超えてはいけません。算出される高さ方向のオフセット量は、工具のパラメータに依存し、穴の選択から独立しています。円柱形の面、円、円弧を選択することができます。
