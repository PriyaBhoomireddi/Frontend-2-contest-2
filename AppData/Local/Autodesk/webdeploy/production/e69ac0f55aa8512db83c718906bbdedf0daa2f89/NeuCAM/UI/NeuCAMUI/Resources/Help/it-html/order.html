Specifica l'ordinamento delle operazioni nel pattern.

Mantieni ordine: esegue la lavorazione di tutte le operazioni in ogni istanza del pattern prima di passare alla successiva.

Ordina per operazione: esegue la lavorazione di tutte le occorrenze di ciascuna operazione del pattern prima di passare all'operazione successiva.

Ordina per utensile: esegue la lavorazione di tutte le operazioni nel pattern che utilizzano l'utensile corrente prima di cambiare utensile.
