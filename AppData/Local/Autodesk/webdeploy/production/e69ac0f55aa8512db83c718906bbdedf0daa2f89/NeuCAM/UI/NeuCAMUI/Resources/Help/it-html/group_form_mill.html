Utensile di forma consente di creare un utensile di forma personalizzata, utilizzando un disegno CAD o uno schizzo importato. La maggior parte dei produttori di utensili dispone nei propri siti Web dei disegni CAD.

La boundary deve essere 2D, piana e chiusa. Le linee spezzate, come le linee di costruzione, non sono consentite.

Dopo che è stato creato, l'utensile viene salvato nel documento corrente nella libreria utensili. È possibile aprire la libreria utensili per aggiungere ulteriori parametri di taglio o descrizioni. Spostare l'utensile in una libreria locale o cloud qualsiasi per l'utilizzo futuro.
