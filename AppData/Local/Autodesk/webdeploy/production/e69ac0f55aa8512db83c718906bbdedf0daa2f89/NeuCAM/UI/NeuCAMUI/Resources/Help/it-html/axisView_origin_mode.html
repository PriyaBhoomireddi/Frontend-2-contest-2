Il menu a discesa Origine offre le seguenti opzioni per individuare l'asse rotativo:

Setup origine WCS: utilizza il sistema di coordinate globali (WCS) del pezzo di lavoro del setup corrente per l'asse rotativo.

Origine modello: utilizza l'origine del sistema di coordinate (WCS) della parte corrente per l'asse rotativo.

Punto selezionato: consente di selezionare un vertice o uno spigolo per l'asse rotativo.
