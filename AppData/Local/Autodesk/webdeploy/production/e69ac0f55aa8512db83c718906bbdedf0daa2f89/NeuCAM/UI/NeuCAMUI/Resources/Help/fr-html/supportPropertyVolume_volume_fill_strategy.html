Définit les directions dans lesquelles les contours extérieurs des supports de volume seront remplis: dans les directions X et Y, dans la direction Y uniquement, ou aucun remplissage.

  • L'équivalent de Uniquement dans X peut être créé à l'aide de l'option Uniquement dans Y, et en définissant l'option Rotation selon Z sur 90 degrés.
  • Non disponible pour le remplissage creux.
