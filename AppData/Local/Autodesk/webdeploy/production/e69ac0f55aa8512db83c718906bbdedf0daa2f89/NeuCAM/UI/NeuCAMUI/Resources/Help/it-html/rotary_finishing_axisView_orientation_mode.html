Specifica l'orientamento dell'asse rotativo.

Il menu a discesa Orientamento fornisce le seguenti opzioni per impostare l'orientamento dell'asse rotativo:

Setup asse X: utilizza l'asse X del sistema di coordinate globali (WCS) del pezzo di lavoro del setup corrente come asse rotativo.

Setup asse Y: utilizza l'asse Y del sistema di coordinate globali (WCS) del pezzo di lavoro del setup corrente come asse rotativo.

Setup asse Z: utilizza l'asse Z del sistema di coordinate globali (WCS) del pezzo di lavoro del setup corrente come asse rotativo.

Orientamento modello - Utilizza il sistema di coordinate (WCS) della parte corrente come l'asse rotativo.

Selezionare asse rotativo - Selezionare una faccia piana per impostare l'asse rotativo perpendicolare a essa o selezionare una faccia circolare per impostare il centro della faccia come il punto attraverso cui passa l'asse rotativo.

Seleziona sistema di coordinate: imposta un orientamento specifico dell'asse rotativo per questa operazione da un sistema di coordinate definito dall'utente (UCS) nel modello. Utilizza l'origine e l'orientamento del sistema di coordinate esistente. Utilizzare questa opzione se il modello non contiene un punto e un piano adatti per l'operazione.
