Activez cette option pour générer l'ébauche en tant que cycle fixe pour votre contrôle.

Cela peut réduire la quantité de code généré pour le programme CN. Cela limite également certaines options avancées pour la trajectoire d'outil Ébauche de profil, car la plupart des cycles fixes ne créent qu'une coupe de limite standard.

Remarque: cette opération peut nécessiter de modifier le post-processeur de manière spécifique par rapport au contrôle de votre machine.
