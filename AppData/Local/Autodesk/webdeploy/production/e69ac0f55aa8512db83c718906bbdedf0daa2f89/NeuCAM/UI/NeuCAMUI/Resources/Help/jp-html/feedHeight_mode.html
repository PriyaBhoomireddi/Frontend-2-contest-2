送り高さでは、工具がパーツへと進入するための送り/切込み速度に切り替える前に、早送りで移動する高さを設定します。送り高さは、トップ より上になるように設定する必要があります。ドリル操作では、この高さを初期送り高さおよび退避ペック高さとして使用します。送り高さとそれ以降のオフセット量を組み合わせて使用することで、送り高さが決定します。

送り高さ
  • 移動高さ: 移動高さ からの相対オフセット値です
  • 退避高さ: 退避高さ からの相対オフセット値です。
  • 無効: 送り高さ を無効にすると、工具は進入動作まで早送りで移動する事になります。
  • トップ高さ: トップ高さ からの相対オフセット値です。
  • ボトム高さ: ボトム高さ からの相対オフセット値です。
  • モデル トップ: モデル トップ からの相対オフセット値です。
  • モデル ボトム: モデル ボトム からの相対オフセット値です。
  • ストック トップ: ストック トップ からの相対オフセット値です。
  • ストック ボトム: ストック ボトム からの相対オフセット値です。
  • 選択された輪郭: モデル上で選択された 輪郭 からの相対オフセット値です。
  • 選択: モデル上で選択された 点(頂点)、エッジ、または 面 からの相対オフセット値です。
  • 原点(絶対): セットアップ または指定した操作の 工具方向 で定義された 原点 からの絶対オフセット値です。
