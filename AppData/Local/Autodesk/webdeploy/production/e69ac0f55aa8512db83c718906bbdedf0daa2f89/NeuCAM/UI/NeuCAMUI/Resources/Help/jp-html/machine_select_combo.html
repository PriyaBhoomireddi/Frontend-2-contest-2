[マシン]プルダウンを使用すると、以前に使用したマシンを選択したり、[なし] を選択して選択マシンを削除することができます。マシン タイプを選択すると、使用可能な加工法が定義されます。

[選択]をクリックすると、使用可能なマシン タイプが表示されます。加工プロセスを選択すると、マシンのリストがフィルタリングされます。

マシン タイプを選択すると、使用可能な加工法が定義されます。

ミル - 回転スピンドルと 2 ～ 5 軸のモーションを備えたマシン。

旋盤 - 2 軸旋盤およびミルターン設定を備えた旋盤。

切断 - ウォータージェット、レーザー、またはプラズマのマシン。

積層 - プラスチックまたは金属を使用した 3D プリンティング。
