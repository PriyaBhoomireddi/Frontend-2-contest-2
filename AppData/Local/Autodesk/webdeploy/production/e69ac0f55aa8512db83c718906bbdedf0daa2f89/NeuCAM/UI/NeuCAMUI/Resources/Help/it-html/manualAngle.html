Specifica la direzione per tutte le passate.

Una direzione di passaggio manuale di 0 gradi è parallela all'asse X del sistema di coordinate globali. Una direzione di passaggio di 90 gradi è parallela all'asse Y del sistema di coordinate globali.

L'uso della direzione di passaggio manuale anziché di quella automatica è da preferirsi, ad esempio, nel caso in cui la direzione automatica generi passate che finiscono contro una parete: così facendo, l'utensile tocca più volte la parete lasciando segni indesiderati.
