Specifica l'angolo di rastremazione delle pareti.

La definizione di un angolo di inclinazione; può essere utilizzato per lavorare feature con una strategia 2D che richiederebbero altrimenti una strategia 3D.

NOTA - L'angolo di inclinazione NON è gestito dalla geometria del modello, pertanto è possibile che un errore di inserimento dell'angolo di inclinazione possa influire sulla lavorazione finita.
